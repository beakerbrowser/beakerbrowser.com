It's time for a better Web

The old Web depends on data silos run by big businesses, which takes away personal control, privacy, and freedom.

Beaker's peer-to-peer Web enables free publishing for all, with open data, private sharing, and no silos.

Don't just browse the Web. Build it!

Beaker is the world's first Web browser to include built-in publishing. Create websites at the click of the button, and start sharing with your audience.

Independent hosting

Beaker is equipped with tools for creating and hosting websites directly from the browser. Beaker uses the Dat peer-to-peer network to host your website's files, so you can share your projects completely independently and without worrying about hosting fees.

Secret filesharing

Files published on the Dat network can only be decrypted by people who know the files' secret URL. This enables truly secret sharing in Beaker, where you can share files privately without ever revealing the files to a third-party hosting service.

Versioned URLs

Every change that's published to a Dat peer-to-peer website is added to its history log. With Beaker, you can a view a website at any point in its history at links like dat://beakerbrowser.com+42.

Live reloading

With builtin tools for live reloading, developing peer-to-peer websites and applications with Beaker is a breeze. While live reloading is on, Beaker will automatically reflect your work on your device.

Offline sync

Save a website for offline use, and as soon as you're back online, Beaker will sync any changes that were published in the background.

New Web APIs

Beaker introduces new Web APIs for building peer-to-peer websites and applications with the Dat protocol.

Mission

We believe in a better Web. Beaker is an experiment for demonstrating the features we believe the Web platform should adopt. Our goal is not to create a “new Web”, but to improve the Web we have. We don’t want to replace existing browsers; we want to change them.
