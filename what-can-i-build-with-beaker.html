What can I build with Beaker?

Beaker adds APIs for downloading and publishing files on the network. Here's a few examples of what you can do:

Self-hosted Blogging

Generate your site using a popular blogging tool, or write a Beaker app to generate in-browser, then publish from your device.

Social Feeds

Write and aggregate feed-files over the network. Use existing standards such as RSS and Atom, or write your own with JSON or YAML.

Integrated Editor

Create a files editor with the Dat API. Write the HTML in one tab, and view your site in the other. Serve the files from the browser when its ready to go.

{% comment %}
Personal Search

Use the SQLite API to index content from the P2P network and provide full-text search.
